## ⚛️ 项目概述

**维度探索**是一个基于**弦理论、M 理论与现代宇宙学**核心框架构建的**交互式高维宇宙科普系统**，旨在打破抽象数学公式与大众认知的壁垒，以可视化、沉浸式、逻辑自洽的方式，带领观测者完成从 0 维奇点到 11 维时空之巅的完整认知跃迁，深度解构空间、时间、平行宇宙与万物理论的底层物理逻辑。

在量子引力与万有理论的探索前沿，高维空间早已不是科幻设定，而是理论物理数学自洽性的必然结果。但对于三维时空的宏观观测者而言，高维的抽象概念始终难以触达。本项目以思想实验为锚点、以严谨物理理论为根基、以交互式体验为载体，将前沿理论物理的核心成果，转化为可感知、可推演、可理解的科普内容，让每一个对宇宙终极真相抱有好奇的人，都能完成一次跨越十一维度的认知旅行。

## 🚀 核心特性

## 🎮 互动功能

### 1. 维度模拟器
- **位置**: 首页 Hero 区域
- **功能**: 拖动滑块实时切换 0-11 维可视化
- **特点**: 每个维度都有独特的动画效果

### 2. 知识测验
- **位置**: 互动区域
- **题型**: 5 道选择题，涵盖核心概念
- **反馈**: 即时显示答案和详细解释

### 3. 维度匹配游戏
- **位置**: Hero 区域
- **玩法**: 限时 60 秒，匹配维度编号与名称
- **关卡**: 难度递增，支持多关卡

### 4. 科幻百科（v6.0 新增）
- **位置**: 页面底部区域
- **内容**: 外星文明、科技树、未来情景、星际政治、宇宙历史
- **交互**: 标签导航、卡片详情、弹窗展示

### 5. 理论卡片
- **位置**: 理论区域
- **交互**: 点击卡片查看详细理论
- **内容**: 6 大核心理论深度解析

## 🛠️ 开发指南

### 添加新维度
编辑 `js/data/dimensions.js`，添加数据对象即可。

### 添加新理论
1. 在 `js/data/theories.js` 添加理论数据
2. 在 HTML 中添加对应卡片

### 自定义样式
编辑 `css/variables.css` 修改主题色。

### 扩展游戏
在 `js/modules/games.js` 中添加新的游戏类。

详细开发文档请查看 [ARCHITECTURE.md](./ARCHITECTURE.md)
