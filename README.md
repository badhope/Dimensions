## 📖 项目简介

**维度探索** 是一个单页面应用，旨在以生动、直观的方式向公众普及高维物理学概念。从零维的"点"到十一维的"M理论之巅"，本项目通过精心设计的 CSS 动画、形象的比喻和交互式测验，将晦涩难懂的物理学理论转化为引人入胜的视觉体验。

### ✨ 核心亮点

- **科幻美学设计**：深空背景配合流星雨动画，营造沉浸式探索氛围。
- **纯前端实现**：无需后端服务器，单个 HTML 文件即可运行。
- **分层架构**：代码结构清晰，数据、逻辑、UI 分离，易于维护。
- **响应式布局**：完美适配桌面端与移动端设备。

---

## 🚀 功能特性

### 1. 视觉体验
- **🌠 动态流星雨背景**：基于 Canvas 的高性能流星与星空渲染。
- **🔮 维度可视化**：每个维度都有独特的 CSS 动画（如 3D 旋转立方体、时间涟漪、高维轨道运动）。
- **🎨 玻璃拟态 UI**：现代化的毛玻璃效果卡片，层次分明。

### 2. 内容深度
- **📚 详细解读**：涵盖 0-11 维度的定义、形象比喻、实例和趣味事实。
- **🧠 深度分析**：可展开的深度分析板块，连接弦理论、M理论等前沿物理。
- **🏛️ 理论基石**：专门的模块介绍卡拉比-丘流形、全息原理等六大理论支柱。

### 3. 交互功能
- **🧪 知识测验**：5道精选题目，测试对维度的理解，即时反馈。
- **🧭 快速导航**：桌面端侧边栏与移动端弹窗，一键直达任意维度。
- **🌓 视觉模式切换**：增强视觉模式，提供不同的色彩体验。
- **📜 阅读进度条**：顶部进度条实时显示阅读进度。
