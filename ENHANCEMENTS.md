## 🎮 4. 新游戏模块

### 4.1 维度拼图挑战 (`js/modules/advancedGames.js`)
益智类维度匹配游戏：

**游戏玩法：**
1. 左侧显示随机排序的维度卡片
2. 右侧显示需要放置的位置
3. 拖拽维度卡片到正确位置
4. 正确放置得 10×关卡分
5. 错误放置扣 5 秒时间
6. 限时 120 秒完成关卡

**特色功能：**
- 🧩 拖拽匹配机制
- ⏱️ 时间限制挑战
- 📈 关卡难度递增（每关增加 2 个维度）
- 💡 提示系统（扣 10 秒时间）
- 🎯 移动端点击选择支持
- 🏆 分数排行榜（集成到进度追踪）

**游戏机制：**
- 时间奖励：剩余时间×2 加分
- 连击系统：连续正确额外加分
- 关卡递进：完成一关自动进入下一关

### 4.2 维度建造者 (`js/modules/advancedGames.js`)
资源管理和策略建造游戏：

**游戏玩法：**
1. 收集三种资源：能量⚡、物质🔮、弦🧵
2. 建造不同维度的结构
3. 每个维度有不同的资源消耗
4. 建造一定数量后升级
5. 升级获得奖励资源

**资源系统：**
```
维度建造成本 = 维度数 × 系数
- 能量：维度数 × 10
- 物质：维度数 × 5
- 弦：维度数 ÷ 2（向上取整）
```

**特色功能：**
- 🏗️ 策略性资源管理
- 📊 实时资源显示
- ⬆️ 等级系统（每建造 3 个维度升一级）
- 🎁 升级奖励（能量 +50, 物质 +25, 弦 +5）
- 🎨 可视化建造进度
- 🔍 可建造项智能筛选（根据当前资源）

**游戏目标：**
- 建造所有 12 个维度
- 达到最高等级
- 优化资源使用效率

## 🔊 5. 音效系统

### 5.1 Web Audio API 音效管理器 (`js/modules/sound.js`)
完整的合成音效系统：

**UI 音效：**
- `playClick()` - 按钮点击 (800Hz 正弦波)
- `playHover()` - 悬停反馈 (1200Hz 正弦波)
- `playSuccess()` - 成功和弦 (C-E-G)
- `playError()` - 错误提示 (150Hz 锯齿波)
- `playAchievement()` - 成就解锁 (上升音阶)

**游戏音效：**
- `playCorrectMatch()` - 正确匹配
- `playWrongMatch()` - 错误匹配
- `playLevelComplete()` - 关卡完成
- `playGameOver()` - 游戏结束
- `playDimensionWarp()` - 维度跃迁（滑音效果）
- `playCardFlip()` - 卡片翻转
- `playResourceCollect()` - 资源收集
- `playBuildComplete()` - 建造完成

**技术特性：**
- 使用 Web Audio API 实时合成
- 无需外部音频文件
- 极低内存占用
- 音量控制（0-100%）
- 一键开关音效
- 浏览器兼容性检测
- 自动恢复 AudioContext

**音效设计：**
- 正弦波 - 清脆 UI 反馈
- 锯齿波 - 错误/警告
- 三角波 - 成就/升级
- 和弦 - 成功/完成
- 滑音 - 特殊效果
